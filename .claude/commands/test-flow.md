사이트의 주요 플로우를 점검해줘.

## 테스트 시나리오

### 1. 회원가입 플로우
- 로그인 페이지 → 소셜 로그인 → 추가정보기입(아이디+전화번호) → 홈
- 이미 가입된 사용자 로그인 → 바로 홈

### 2. 시험 응시 플로우
- 홈 → 카테고리 선택 → 시험 선택 → 시험 시작 → 문제 풀기 → 제출 → 결과
- 시험 중단 → 기록 삭제 확인

### 3. 관리자 플로우
- 시험 관리 → 시험 생성(년도/회차) → 문제 관리 → 문제 추가
- 회원 관리 → 권한 변경

### 4. 마이페이지 플로우
- 응시 기록 조회 → 오답노트
- 프로필 수정 → 회원 탈퇴

각 플로우에서 관련된 API 라우트와 페이지 코드를 읽고, 누락되거나 깨진 부분이 있는지 확인해줘.
