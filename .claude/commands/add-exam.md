새 시험(년도/회차)을 추가하는 작업을 도와줘.

## 사용자에게 물어볼 것
1. 카테고리 (전기기초/전기기능사/전기산업기사/전기기사)
2. 년도 (예: 2024)
3. 회차 (예: 1, 2, 3, 4)
4. 한 번에 여러 회차를 추가할지 (예: 2024년 1~4회 한꺼번에)

## 작업 절차
1. DB에 시험 존재 여부 확인
2. 관리자 API (`POST /api/admin/exams`)로 시험 생성
3. 과목은 같은 카테고리 기존 시험에서 자동 복사됨
4. 생성 후 시험 관리 페이지에서 확인 안내
5. 문제 추가 방법 안내 (관리자 > 시험 관리 > 문제관리 링크)

## 일괄 추가 시
Node 스크립트로 pg Pool을 사용해 직접 INSERT (기존 시험 추가 스크립트 패턴 참고)
