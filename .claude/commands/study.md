기술적인 내용을 비유를 들어서 쉽게 설명해주고, study.md에 기록해줘.

## 설명 방식
- 일상생활 비유를 활용 (식당, 냉장고, 주방장 등 기존 비유 체계 유지)
- 전문 용어를 먼저 쓰고 괄호 안에 비유 설명
- 코드 예시와 비유를 함께 제시
- 핵심만 간결하게, 길어지지 않게

## study.md 작성 형식
```
### [주제]
- 한 줄 요약
- 비유 설명
- 실제 코드에서의 적용
```

## 기존 비유 체계
- 프로젝트 = 식당
- 데이터베이스 = 냉장고
- Prisma = 주방장
- API = 주문서
- 미들웨어 = 입구 직원
- JWT = 도장
