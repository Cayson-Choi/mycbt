CAYSON(mycbt) 사이트의 SEO 최적화 상태를 점검하고 개선해줘.

## 점검 항목

### 1. 메타데이터
- layout.tsx의 metadata 객체 확인 (title, description, keywords, og tags)
- 각 페이지별 동적 메타데이터 설정 여부

### 2. 시맨틱 HTML
- 적절한 heading 구조 (h1 → h2 → h3)
- main, article, section, nav 태그 사용

### 3. 성능 관련 SEO
- 이미지 alt 속성
- next/image 사용 여부
- Core Web Vitals에 영향주는 요소

### 4. robots.txt / sitemap
- public/robots.txt 존재 여부
- sitemap.xml 자동 생성 설정

### 5. 구조화 데이터
- JSON-LD 스키마 적용 가능한 페이지 식별

점검 결과를 우선순위별로 정리하고, 가장 효과 큰 것부터 수정 제안해줘.
타겟 키워드: 전기기능사 기출문제, 전기기사 CBT, 전기산업기사 모의고사, 전기 자격증 시험
