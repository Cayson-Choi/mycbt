CAYSON(mycbt) 프로젝트의 기능이나 개선 아이디어를 브레인스토밍해줘.

## 진행 방식
1. 사용자가 제시한 주제에 대해 아이디어를 최소 5개 이상 제시
2. 각 아이디어에 대해:
   - 한 줄 설명
   - 구현 난이도 (쉬움/보통/어려움)
   - 예상 효과 (높음/중간/낮음)
   - 우선순위 추천

## 브레인스토밍 영역 예시
- 사용자 경험 개선
- 새로운 기능
- 수익화 방안
- 마케팅/성장 전략
- 기술 부채 해소

핵심은 실현 가능한 아이디어 위주로. 뜬구름 잡는 제안은 하지 말 것.
현재 기술 스택(Next.js + Prisma + Neon + Vercel)에서 구현 가능한 것 위주로.
