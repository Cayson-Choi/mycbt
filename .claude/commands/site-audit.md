전기짱(mycbt) 사이트 전체를 종합 점검해줘.

## 점검 범위

### 1. 기능 완성도
- 로그인 4가지 방식 (Google/Kakao/Naver/Email) 정상 동작 여부
- 회원가입 → 시험 응시 → 채점 → 결과 확인 전체 플로우
- 관리자 기능 (시험 관리, 문제 관리, 회원 관리)
- 마이페이지 (응시 기록, 오답노트, 프로필 수정, 탈퇴)

### 2. UI/UX
- 모바일 반응형 깨지는 곳 (Tailwind 클래스 점검)
- 다크모드 누락된 곳 (dark: 클래스 빠진 곳)
- 로딩 상태 표시 누락
- 에러 상태 처리 누락

### 3. 데이터 정합성
- DB에 orphan 데이터 없는지
- API 응답 형식 일관성 (snake_case 통일)
- question_type 값 일관성 (MULTIPLE_CHOICE 통일)

### 4. 코드 품질
- 사용되지 않는 파일/컴포넌트 (_backup 파일 등)
- 콘솔 에러/경고
- TypeScript 타입 오류

발견된 문제를 심각도별로 분류하고, 수정 우선순위를 매겨줘.
