전기짱(mycbt) UI/UX 디자인 작업을 해줘.

## 디자인 원칙
- 상용 서비스 수준의 고급스러운 디자인
- AI가 만든 것 같지 않은, 전문 디자이너가 작업한 느낌
- 이모지/이모티콘 사용 금지
- Tailwind CSS 기반, 다크모드 반드시 지원
- 모바일 퍼스트 반응형

## 컬러 시스템
- 메인: blue-600 (다크: blue-400)
- 성공: green-600, 오류: red-600, 경고: amber-600
- 배경: gray-50 (다크: gray-900)
- 카드: white (다크: gray-800)

## 컴포넌트 스타일
- 카드: rounded-lg/xl, shadow-sm, border dark:border-gray-700
- 버튼: rounded-lg, 적절한 padding, hover/disabled 상태
- 입력: border dark:border-gray-600, focus:ring-2 focus:ring-blue-500

사용자가 요청한 페이지/컴포넌트를 위 원칙에 맞게 디자인하고 구현해줘.
