코드베이스를 정리해줘.

## 정리 항목

### 1. 사용되지 않는 파일
- *_backup* 파일들 확인 (이미 .gitignore에 포함되어 있는지)
- 빈 파일, 임시 파일 확인
- 프로젝트 루트의 불필요한 파일 (*.tmp, *.log 등)

### 2. import 정리
- 사용되지 않는 import 제거
- 사용되지 않는 변수/함수 제거

### 3. console.log 정리
- 디버그용 console.log 제거 (console.error는 유지)

### 4. 타입 개선
- `any` 타입 사용 부분 식별 (수정은 사용자 확인 후)

### 5. .gitignore 점검
- 불필요한 파일이 추적되고 있지 않은지

삭제/수정 전에 목록을 보여주고 확인 받을 것. 무작정 삭제하지 말 것.
