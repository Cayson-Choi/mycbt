# 문제 코드 체계 (Question Code System)

## 코드 형식
```
[분야(4자리)]-[등급(1자리)]-[과목(2자리)]-[번호(3자리)]

예시: ELEC-F-TH-001
- ELEC: 전기 분야
- F: 기능사 등급
- TH: 전기이론 과목
- 001: 문제 번호
```

---

## 1. 분야 코드 (Field Code)

| 코드 | 분야명 | 설명 |
|------|--------|------|
| ELEC | 전기 | 전기기능사, 전기산업기사, 전기기사 |
| ELCS | 전기공사 | 전기공사기능사, 전기공사산업기사, 전기공사기사 |
| ELET | 전자 | 전자기능사, 전자산업기사, 전자기사 |
| **FIEE** | **소방설비(전기) 일반형** | 소방설비산업기사(전기), 소방설비기사(전기) |
| **FIEP** | **소방설비(전기) 과정평가형** | 소방설비산업기사(전기) 과정평가형 |
| **FIME** | **소방설비(기계) 일반형** | 소방설비산업기사(기계), 소방설비기사(기계) |
| **FIMP** | **소방설비(기계) 과정평가형** | 소방설비산업기사(기계) 과정평가형 |
| HYDR | 공유압 | 공유압기능사 등 (추후) |
| MECH | 기계 | 기계기능사, 기계산업기사, 기계기사 (추후) |
| INFO | 정보처리 | 정보처리기능사, 정보처리산업기사, 정보처리기사 (추후) |

## 4. 코드 예시

### 전기 계열
```
ELEC-F-TH-001  전기기능사 - 전기이론 - 1번
ELEC-I-TH-001  전기산업기사 - 전기이론 - 1번
ELEC-E-TH-001  전기기사 - 전기이론 - 1번

ELCS-F-TH-001  전기공사기능사 - 전기이론 - 1번
ELCS-I-TH-001  전기공사산업기사 - 전기이론 - 1번
ELCS-E-TH-001  전기공사기사 - 전기이론 - 1번
```

### 전자 계열

```
과정평가형:

FIMP-I-FR-001  소방설비산업기사(기계) 과정평가형 - 소방원론 - 1번
```

## 5. 자동 생성 규칙

1. **question_code가 비어있으면** 자동 생성
2. **자동 생성 방법:**
   - 시험명 → 분야 + 등급 코드
   - 과목명 → 과목 코드
   - 해당 시험/과목의 마지막 번호 + 1
3. **예시:** 전기기능사 전기이론의 마지막 문제가 ELEC-F-TH-050이면, 다음은 ELEC-F-TH-051
