# 수식 표기 가이드

## 개요
문제, 선택지, 해설에서 수학 및 과학 기호를 올바르게 표시하기 위한 HTML 태그 사용법입니다.

## 지원 태그

### 1. 아래 첨자 (Subscript) - `<sub>`
작은 글자를 아래에 표시합니다.

**사용 예시:**
```
ε<sub>1</sub> > ε<sub>2</sub>인 경우
```
**화면 표시:**
ε₁ > ε₂인 경우

**전기 관련 예시:**
- `V<sub>1</sub>` → V₁ (전압 1)
- `I<sub>2</sub>` → I₂ (전류 2)
- `R<sub>total</sub>` → Rₜₒₜₐₗ (전체 저항)
- `C<sub>1</sub>` → C₁ (커패시터 1)

### 2. 위 첨자 (Superscript) - `<sup>`
작은 글자를 위에 표시합니다 (제곱, 세제곱 등).

**사용 예시:**
```
x<sup>2</sup> + y<sup>2</sup> = r<sup>2</sup>
```
**화면 표시:**
x² + y² = r²

**전기 관련 예시:**
- `P = I<sup>2</sup>R` → P = I²R (전력 공식)
- `10<sup>3</sup>Ω` → 10³Ω (킬로옴)
- `V<sup>2</sup>/R` → V²/R

### 3. 분수 표기 - `<frac>`
분수를 제대로 표시합니다.

**사용 예시:**
```
<frac>1/5</frac>
```
**화면 표시:**
¹⁄₅

**전기 관련 예시:**
- `<frac>1/2</frac>πfL` → ¹⁄₂πfL
- `V = <frac>Q/C</frac>` → V = Q⁄C
- `<frac>V<sub>1</sub>/V<sub>2</sub></frac>` → V₁⁄V₂

### 4. 복합 사용
```
전압 V<sub>1</sub><sup>2</sup>을 R로 나눈 값
```
**화면 표시:**
전압 V₁²을 R로 나눈 값

```
임피던스 <frac>1/ωC</frac>에서 ω는 각속도
```
**화면 표시:**
임피던스 ¹⁄ωC에서 ω는 각속도

## 사용 시 주의사항

### ✅ 올바른 사용
```
ε<sub>1</sub> > ε<sub>2</sub>일 때
```

### ❌ 잘못된 사용
```
ε1 > ε2일 때  (태그 없이 숫자만)
1/5  (태그 없이 슬래시만)
```

### ✅ 올바른 사용 (분수)
```
<frac>1/5</frac>  (분수 태그 사용)
```

## 문제 등록 예시

### JSON 형식으로 등록할 때:
```json
{
  "question_text": "ε<sub>1</sub> > ε<sub>2</sub>인 경우 발별력이 적용한다.",
  "choice_1": "ε<sub>1</sub> > ε<sub>2</sub>인 경우 음압력이 적용한다.",
  "choice_2": "ε<sub>1</sub> < ε<sub>2</sub>인 경우 발별력이 적용한다.",
  "choice_3": "전압의 제곱 V<sup>2</sup>에 비례한다.",
  "choice_4": "저항 R<sub>total</sub>에 반비례한다.",
  "explanation": "ε<sub>1</sub>이 ε<sub>2</sub>보다 클 때 발별력이 작용합니다. 임피던스는 <frac>1/ωC</frac>입니다."
}
```

### 관리자 페이지에서 직접 입력할 때:
문제 내용 입력란에 HTML 태그를 직접 입력하면 됩니다:
```
ε<sub>1</sub> > ε<sub>2</sub>인 경우 발별력이 적용한다.
```

## 자주 사용되는 기호

| 입력 | 표시 | 설명 |
|------|------|------|
| `ε<sub>1</sub>` | ε₁ | 유전율 1 |
| `μ<sub>0</sub>` | μ₀ | 진공 투자율 |
| `I<sub>rms</sub>` | Iᵣₘₛ | 실효값 전류 |
| `10<sup>6</sup>` | 10⁶ | 메가 (백만) |
| `θ<sub>1</sub>` | θ₁ | 각도 1 |
| `ω<sub>0</sub>` | ω₀ | 고유 각주파수 |
| `<frac>1/2</frac>πfL` | ¹⁄₂πfL | 유도 리액턴스 |
| `<frac>1/ωC</frac>` | ¹⁄ωC | 용량 리액턴스 |

## 보안 안내
- 시스템에서는 `<sub>`, `<sup>`, `<frac>` 태그만 허용됩니다
- 다른 HTML 태그는 자동으로 제거되므로 안전합니다
- XSS 공격 방지를 위해 태그 속성은 모두 제거됩니다
- `<frac>` 태그는 내부적으로 `<sup>`와 `<sub>`로 변환되어 표시됩니다

## 미리보기 기능
관리자 페이지의 대량 업로드 기능에서 미리보기를 통해 수식이 제대로 표시되는지 확인할 수 있습니다.
